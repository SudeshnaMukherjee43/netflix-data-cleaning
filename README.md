# Netflix Data Cleaning Project

## Objective
The goal of this task is to clean and preprocess the Netflix dataset by handling missing values, correcting data types, and preparing the dataset for analysis.

## Steps Performed
- Identified missing values in columns
- Filled missing values using appropriate placeholders
- Converted date_added column to datetime format
- Removed inconsistencies and ensured clean data
- Saved the cleaned dataset as a new CSV file

## Tools Used
- Python
- Pandas
- VS Code

## Output
- cleaned_netflix_data.csv
